PROJET BBQ PROFESSEUR
=====================

Vous pouvez démarrer le logiciel en double cliquant sur l'exécutable "ProjetBBQProf.exe".
La première fois, celui-ci se fermera tout seul et créera un dossier dans lequel il faudra ajouter des salles ainsi qu'un fichier de configuration.

Pour éditer la configuration ou les salles, il faut fermer le logiciel.

Si le fichier ne se créé pas, veuillez être sur qu'il est possible pour le logiciel d'écrire dans le dossier d'installation;
Sinon : clic droit -> exécuter en mode administrateur.

Pour voir la liste des problèmes courants, veuillez consulter la FAQ (https://github.com/Skyost/ProjetBBQ/wiki/FAQ).
